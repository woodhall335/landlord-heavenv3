# Previews image naming

## Naming convention
- Use kebab-case.
- Prefer `.webp`; allow `.png` only if needed.
- Paths include the document type folder:
  `/images/previews/{product}/{jurisdiction}/{notice-type}/{filename}.webp`
- Filenames should be URL-safe (no spaces); prefer lowercase + hyphens.
- The UI derives preview titles from filenames, so keep them descriptive.

## Examples

- `/images/previews/notice-only/england/section21/section-21-form-6a-preview.webp`
- `/images/previews/notice-only/england/section21/service-instructions-preview.webp`
- `/images/previews/notice-only/england/section21/validity-checklist-preview.webp`

- Keep filenames stable and descriptive (used for image SEO + sitemap later).
- Use keywords in filenames (section 21 form 6a, section 8 form 3, notice to leave prt, etc.).
